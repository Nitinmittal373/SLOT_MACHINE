# 🎰 Slot Machine Game

A simple command-line Slot Machine game built with Python.

This project was created as a **Python revision and practice exercise** while refreshing core Python concepts such as functions, loops, dictionaries, lists, user input, and randomization. The implementation follows a YouTube tutorial, and I built it myself as part of my learning journey.

## Concepts Practiced

- Functions
- Loops
- Dictionaries
- Lists
- Random module
- Input validation
- Program structure
